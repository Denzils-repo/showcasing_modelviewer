Google Web Component

Embedded 3D, Democratized

Google's <model-viewer> is a declarative web component that makes displaying interactive 3D models on websites as easy as writing an image tag. It is responsive, accessible, and runs smoothly on all major platforms.

How Does It Work?

Unlike complex WebGL/Three.js custom setups, Google Model Viewer works completely natively in your HTML. Just add the script CDN and write the tag.

The Quick Start Syntax

To use it, copy the script block below into your HTML document, then place the component tag in your body. It will automatically load the model, generate interactive orbit controls, configure responsive canvas bounds, and handle touch gestures automatically.

<!-- Import the module library --> <script type="module" src="https://ajax.googleapis.com/ajax/libs/model-viewer/3.5.0/model-viewer.min.js"></script> <!-- Use the component tag --> <model-viewer src="./chest.glb" alt="Treasure Chest Model" camera-controls auto-rotate shadow-intensity="1.5" ></model-viewer>

Essential Properties

Attribute Type / Values Description
src URL String Path to the 3D asset (requires GLTF/GLB).
camera-controls Boolean Enables mouse & touch orbit, zoom, and panning.
auto-rotate Boolean Slowly spins the model when there is no interaction.
shadow-intensity Number (0 - 2) Darkness scale of the ground contact shadow.
ar Boolean Enables Augmented Reality mode on supported devices.

Key Features

Designed for the modern web environment with standard browser support, responsive sizing, and high-fidelity rendering built in.

Augmented Reality (AR)

Google Model Viewer is integrated with WebXR and Apple's AR Quick Look, letting users view models placed in their real surroundings on iOS and Android instantly.

A11y Accessibility

Fully compliant with ARIA accessibility guidelines. Users can orbit models via keyboard tab controls and it supports screen-reader descriptions.

Performance & Lazy Loading

Includes poster images, customizable loader layouts, and native lazy-loading settings so your initial page load remains fast and lightweight.

Interactive Playground

Customize properties of the 3D model in real time. Watch how the attributes affect the rendering and copy the generated code instantly.

<model-viewer
  src="./chest.glb"
  camera-controls
  auto-rotate
  shadow-intensity="1"
  shadow-softness="0.5"
  exposure="1"
></model-viewer>

Sources for 3D Models

To use Google Model Viewer, you need 3D models in GLB/GLTF format. Here are the top platforms where you can download free or premium 3D assets.

Sketchfab

Free & Paid GLTF Available Largest Library

Sketchfab is the largest platform for 3D models on the web. It has built-in auto-conversions to GLTF/GLB formats, making it easy to download and use models directly.

Visit Sketchfab

Poly Haven

100% Free (CC0) High Quality GLTF / Textures

Poly Haven offers completely free, high-quality, photorealistic scans, models, HDRI environments, and materials under a CC0 license (public domain).

Visit Poly Haven

CGTrader

Free Filter Commercial Assets Any Format

A massive marketplace of 3D assets geared towards professionals. You can filter by "Free" and choose GLTF format or convert standard formats like FBX/OBJ to GLB.

Visit CGTrader
